# 部署指南：將健康管理系統發布到網路

## 📋 前置準備

1. **GitHub 帳號**：如果還沒有，請先到 [github.com](https://github.com) 註冊
2. **Vercel 帳號**：等等會用 GitHub 帳號直接登入，不需要另外註冊

---

## 🚀 部署步驟

### 步驟 1：在 GitHub 建立新的 Repository

1. 登入 GitHub 後，點擊右上角的 `+` → `New repository`
2. 填寫以下資訊：
   - **Repository name**: `health-tracker`（或妳喜歡的名稱）
   - **Description**: `孕期健康管理系統`
   - **Privacy**: 選擇 `Private`（私人專案，只有妳看得到）
   - ⚠️ **不要勾選** "Add a README file"（我們已經有了）
3. 點擊 `Create repository`

### 步驟 2：推送程式碼到 GitHub

複製 GitHub 顯示的指令，或直接在 PowerShell 執行以下指令：

```powershell
# 1. 進入專案資料夾（如果還沒在裡面的話）
cd "C:\Users\蔡彤雨\Desktop\Antigravity\問問題用\DietTracker"

# 2. 設定遠端 Repository（請替換成妳的 GitHub 使用者名稱）
git remote add origin https://github.com/你的使用者名稱/health-tracker.git

# 3. 推送程式碼
git branch -M main
git push -u origin main
```

> 💡 **提示**：第一次推送時，系統會要求妳登入 GitHub。請使用妳的 GitHub 帳號密碼。

### 步驟 3：在 Vercel 部署

1. 打開瀏覽器，前往 [vercel.com](https://vercel.com)
2. 點擊 `Sign Up` 或 `Log In`
3. 選擇 `Continue with GitHub`（用 GitHub 帳號登入）
4. 授權 Vercel 存取妳的 GitHub
5. 登入後，點擊 `Add New...` → `Project`
6. 在 `Import Git Repository` 頁面：
   - 找到妳剛剛建立的 `health-tracker` repository
   - 點擊 `Import`
7. 在設定頁面：
   - **Project Name**: 可以保持預設或改成妳喜歡的名稱
   - **Framework Preset**: 選擇 `Other`
   - **Root Directory**: 保持 `./`
   - **Build Settings**: 不需要修改
8. 點擊 `Deploy`

### 步驟 4：等待部署完成

- Vercel 會開始自動部署，通常只需要 30 秒到 1 分鐘
- 部署完成後，會看到慶祝畫面 🎉
- 妳會得到一個網址，格式類似：`https://health-tracker-xxxx.vercel.app`

---

## 📱 如何使用

### 在手機上使用

1. 複製 Vercel 給妳的網址（例如：`https://health-tracker-xxxx.vercel.app`）
2. 在手機瀏覽器（Safari 或 Chrome）輸入這個網址
3. 建議將網站「加入主畫面」：
   - **iPhone**: 點擊分享按鈕 → 選擇「加入主畫面」
   - **Android**: 點擊選單 → 選擇「新增至主畫面」

### 在電腦上使用

直接在瀏覽器開啟 Vercel 給妳的網址即可。

## ⚠️ 重要提醒

- **資料儲存位置**：目前資料存在妳的瀏覽器裡（localStorage），所以：
  - 每個裝置/瀏覽器的資料是**獨立**的
  - 清除瀏覽器資料會導致紀錄消失
  - 如果需要多裝置同步，之後可以升級成 Supabase 版本

- **隱私保護**：
  - GitHub Repository 已設為 Private（只有妳看得到程式碼）
  - 但 Vercel 網址是公開的，任何知道網址的人都能開啟
  - 不過資料存在本地，別人開啟只會看到空白的系統

## 🆘 常見問題

**Q: 我忘記 Vercel 給我的網址了怎麼辦？**  
A: 登入 [vercel.com](https://vercel.com)，在 Dashboard 就能看到所有專案的網址。

**Q: 可以改網址嗎？**  
A: 可以！在 Vercel 專案設定裡的 `Domains` 可以設定自訂網域名稱。

**Q: 部署失敗怎麼辦？**  
A: 檢查 Vercel 的部署日誌（Deployment Logs），或回來問我！
