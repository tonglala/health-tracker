# 健康管理系統

一個簡潔優雅的孕期體重與飲食追蹤應用程式。

## 功能特色

- 📊 **體重追蹤**：區分家裡測量與診所測量，提供精準的趨勢分析
- 🍽️ **飲食日記**：記錄每日飲食與熱量，智慧搜尋功能協助查詢食物熱量
- 📈 **視覺化圖表**：基於 IOM 標準的體重增長曲線，清楚掌握健康狀態
- 📱 **響應式設計**：完美支援手機與桌面裝置

## 技術架構

- 純前端應用（HTML + CSS + JavaScript）
- 使用 Chart.js 進行資料視覺化
- LocalStorage 本地資料儲存
- 響應式設計，支援行動裝置

## 本地開發

直接開啟 `index.html` 即可使用。

## 部署

本專案已配置 Vercel 部署，推送到 GitHub 後可自動部署。
